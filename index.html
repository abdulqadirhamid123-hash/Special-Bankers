How this works
  1. Select A/B/C/D. If an answer key exists, your choice will show a green tick for the correct option and a red cross on an incorrect selection. Your selection is saved.
  2. To set the correct answer for a question, click Set Correct (Admin) and then click the correct letter. Use Export Answer Key / Import Answer Key to save or load keys.
  3. Each question is shown as an image. If you provided a matching answer image (filename contains answer or ans), the Show / Hide Answer button will appear.
  4. Use Mark Correct / Mark Incorrect to track progress; this is saved in your browser.
  5. You can Export your progress to a JSON file and Import it later to continue.
  6. Upload these files to a GitHub repo and enable Pages to host it, or open index.html locally.

If some answers don't pair automatically, rename the answer images to include answer or ans (e.g., Q12_answer.png).
